## How to Contribute

We welcome contributions in many forms:

1. **Code**: Implement features, fix bugs, optimize performance
2. **Science**: Theoretical analysis, empirical validation, peer review
3. **Documentation**: Improve guides, add examples, fix typos
4. **Testing**: Run experiments, report results, validate predictions
5. **Funding**: Financial support for research and development

See [FUNDING.md](FUNDING.md) for sponsorship opportunities.

See [CONTRIBUTING.md](CONTRIBUTING.md) (coming soon) for contribution guidelines.
